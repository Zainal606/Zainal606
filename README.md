## 🚀 Featured Project

### 💰 Loan Tracker — Personal Finance Intelligence Dashboard

> Built after giving my loan details to Claude and realising I didn't understand my own finances. One personal finance problem → one deployed cloud product.

[![Live App](https://img.shields.io/badge/🔴%20Live%20App-Streamlit-FF4B4B?style=for-the-badge)](https://apploanproject-yzdbzw6gid9bwul2upareu.streamlit.app/)
[![Repo](https://img.shields.io/badge/📁%20Source%20Code-GitHub-181717?style=for-the-badge&logo=github)](https://github.com/Zainal606/Streamlit_loan_project)

**What it does:**
- Real-time amortization dashboard connected to Azure SQL
- Financial intelligence cards — rate assessment, daily cost, EMI discipline
- 3 payoff scenarios (aggressive / moderate / conservative) with exact dates
- Interactive forecast chart with full amortization schedule
- Payment recording with instant dashboard refresh

**Stack:** `Python` · `Streamlit` · `Azure SQL` · `Plotly` · `pyodbc` · `Streamlit Cloud`
